자바스크립트에서 객체 내부의 속성을 의미한다.
객체는 프로퍼트로 구성된다
프로퍼티는 "key(키)" : "value(값)" 의 형식으로 객체 안의 콤마(쉼표 ,)로 구분되어 할당된다.
key는 속성명 이라고 생각해도 되고 문자열만만만(""없이가능함,띄어쓰기가 있는 경우 필요험) 가능하다 
value는 속성값 이라고 생각해도 되고 어떤 값이든(문자열,숫자,함수,등등등) 상관없다.

property에 접근하는 방법은 2가지가 있다.

변수명.a;    // 'a'
변수명['a']; // 'a'

대괄호([])로 접근 (브라캣 연산자)
점(.) 표기법을 이용한 접근
주로 마침표를 사용하고 대괄호는 속성명에 띄어쓰기가 들어가 있는 경우 사용한다.
간단한 Property일 수도 있고, 배열의 일부이거나 객체의 Method를 호출할 수도 있다.
